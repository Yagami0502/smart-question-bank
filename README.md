# MindForge - 智能学习平台

一个综合性智能学习平台，集成题库系统、词汇学习、文章阅读、间隔重复算法、番茄钟、学习分析等功能。

## 技术栈

| 类别 | 技术 |
|------|------|
| 前端 | React 19 + TypeScript + Vite 5 + TailwindCSS 3 |
| 状态管理 | Zustand |
| 后端 | Express + MySQL 8.0 |
| 部署 | Docker Compose |

## 功能特性

### 1. 题库系统

- **5 种题型**：单选、多选、判断、填空、简答
- **5 种练习模式**：智能练习、新题练习、复习模式、冲刺模式、考试模式
- **多格式导入**：CSV、Excel、Word、文本、AI 智能导入
- **收藏与错题本**：支持收藏题目、自动记录错题
- **笔记与标签**：为题目添加笔记和标签
- **全文搜索**：支持题目内容全文检索

### 2. 词汇学习

- **单词本**：含音标、翻译、短语、例句
- **3 种练习模式**：打字练习、复习模式、听写模式
- **智能/自由学习模式**：基于掌握程度自适应
- **语音合成**：支持美式/英式发音
- **键盘音效**：打字时播放按键音
- **错词本与收藏**：自动记录错误单词

### 3. 文章阅读

- 打字模式和听写模式
- 翻译对照
- 语音播放

### 4. SM-2 间隔重复算法

- 完整的 SM-2 算法实现
- 易度因子、间隔计算、遗忘曲线预测
- 加权随机采样

### 5. 学习工具

- **番茄钟计时器**
- **闪卡复习**
- **学习计划**
- **每日目标**
- **学习提醒**

### 6. 数据分析

- 统计报表（Recharts）
- 学习日历热力图
- 时间线、知识图谱、错误分析

### 7. 成就系统

- 答题数量、连续天数、正确率、连击等成就徽章

### 8. 社交与导出

- 分享卡片
- 数据导出（JSON / CSV / TXT / Markdown）

### 9. 设置

- 学习设置、算法设置、AI 设置、安全设置
- 国际化（中文 / 英文，自动检测）

### 10. 认证与安全

- JWT + refresh token
- bcrypt 密码加密
- 速率限制、Helmet 安全头
- 会话管理

### 11. 管理后台

- 用户管理、内容管理、系统统计

## 项目结构

```
smart-question-bank/
├── src/
│   ├── components/         # React 组件
│   ├── pages/              # 页面组件
│   ├── stores/             # Zustand 状态管理
│   ├── lib/                # 工具库（SM-2 算法等）
│   ├── hooks/              # 自定义 Hooks
│   └── types/              # TypeScript 类型定义
├── server/
│   ├── routes/             # Express API 路由
│   ├── middleware/          # 中间件
│   └── db/                 # 数据库配置
├── docker-compose.yml
├── package.json
└── vite.config.ts
```

## 快速开始

```bash
# 使用 Docker Compose
docker-compose up --build

# 或本地开发
npm install
npm run dev        # 前端开发服务器
npm run server     # 后端服务器
npm run test       # 运行测试
```
