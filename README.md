# MindForge 🧠

**Think. Learn. Conquer.**

MindForge 是一款全方位智能学习平台，集题库刷题、词汇记忆、阅读训练、学习管理于一体，基于 SM2 间隔重复算法驱动个性化学习体验。

## ✨ 核心功能

### 📚 题库系统
- 公共题库浏览与个人题库管理
- 多格式导入（CSV、文本等）
- 五种练习模式：智能复习、学习新题、全部复习、强化训练、模拟考试
- 收藏夹 & 错题本，针对性查漏补缺

### 🌍 词汇学习
- 公共词库与个人词库管理
- 单词练习与拼写测试
- 错词本 & 收藏词管理
- 词汇学习统计与进度追踪

### 📖 文章阅读
- 英语文章阅读训练
- 文章内生词标注与练习

### 🎯 学习工具
- **闪卡速记** — 快速记忆卡片
- **番茄钟** — 专注学习计时
- **学习计划** — 制定个性化学习目标
- **每日目标** — 日常学习任务追踪
- **学习提醒** — 定时复习提醒

### 📊 数据与分析
- 学习统计报告（练习时长、正确率、连续学习天数）
- 学习日历 — 打卡记录可视化
- 学习时间线 — 历史学习轨迹
- 知识图谱 — 知识点关联可视化
- 错题分析 — 薄弱点精准定位

### 👤 个人中心
- 成就系统与勋章
- 学习分享
- 数据导出
- 个性化设置（主题、语言切换）

### 🔐 系统功能
- 用户认证（注册 / 登录）
- 管理员后台（用户管理、内容管理）
- 中英双语支持（i18n）

## 🛠 技术栈

| 分类 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 5 |
| 样式 | TailwindCSS 3 + Liquid Glass UI |
| 状态管理 | Zustand |
| 本地数据库 | Dexie (IndexedDB) |
| 后端 | Node.js + Express + MySQL |
| 图表 | Recharts |
| 国际化 | i18next |
| 图标 | Lucide React |

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动前端开发服务器
npm run dev

# 启动后端服务（需要 MySQL）
cd server
npm install
npm start
```
